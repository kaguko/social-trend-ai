# Crawler Module

Module này thu thập dữ liệu từ mạng xã hội và báo điện tử.

## Nguồn dữ liệu dự kiến
- Facebook public posts/comments (qua Graph API hoặc scraping)
- YouTube comments
- Báo điện tử (VnExpress, Thanh Niên, Tuổi Trẻ)

## Files
- `facebook_crawler.py` — Crawl Facebook
- `youtube_crawler.py` — Crawl YouTube
- `news_crawler.py` — Crawl báo điện tử
- `data_collector.py` — Gom tất cả nguồn
